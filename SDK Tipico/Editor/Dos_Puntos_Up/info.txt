WM_LBUTTONUP: mensaje que se envia cuando se deja de
presionar el boton izquierdo del mouse

WM_LBUTTONUP 
fwKeys = wParam;        // key flags 
xPos = LOWORD(lParam);  // horizontal position of cursor 
yPos = HIWORD(lParam);  // vertical position of cursor 

fwKeys
MK_CONTROL activo si se presiona la tecla control
MK_MBUTTON activo si se presiona el boton medio del mouse
