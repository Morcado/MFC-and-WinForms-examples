MK_LBUTTON              se activa cuando se da click izquierdo

CreatePen: crea un pen logico que tiene especificado el estilo, el tamaño y el color.
El pen puede set seleccionado en un device context usado para dibujar lineas y curvas

HPEN CreatePen( 
  int fnPenStyle,    // estilo
  int nWidth,        // ancho  
  COLORREF crColor   // color
);

fnPenStyle
PS_SOLID     el pen es solido
PS_DASH     el en es con lineas
/***************************************************************************************/
SelectObject: selecciona un objeto para el device context y reemplaza el anterior

HGDIOBJ SelectObject(
  HDC hdc,          // handle to device context
  HGDIOBJ hgdiobj   // handle para objeto
);

hgdiobj: Object, Bitmap, Brush, Font, Pen, Region
